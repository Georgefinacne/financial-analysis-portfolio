## Project 1 — Next plc 3-Statement Financial Model & DCF Valuation

**Tools:** Microsoft Excel  
**Company:** Next plc (LSE: NXT) — FTSE 100 UK Retailer

### What I built:
- Fully integrated 3-statement financial model (Income Statement, 
  Balance Sheet, Cash Flow Statement) with 3 years of historical 
  data and 3 years of projections
- DCF valuation incorporating WACC calculation, free cash flow 
  projections and terminal value
- Sensitivity analysis table showing implied share price across 
  different WACC and terminal growth rate assumptions

### Key outputs:
- Projected revenue growing from £5.7bn to £6.9bn by FY2027E 
  at stable EBITDA margins of 25%
- DCF implied share price compared against live market price of £137.70
- Sensitivity table demonstrating valuation impact of key assumptions

## Project 2 — FTSE 100 Retail Sector Equity Analysis

**Tools:** Python (pandas, yfinance, matplotlib, seaborn)  
**Universe:** Next, Tesco, M&S, JD Sports, Sainsbury's

### What I built:
- Python script to extract 12 months of live daily price data 
  for 5 FTSE 100 retailers using yfinance
- Calculated annualised return, volatility and Sharpe ratio 
  for each stock
- Generated a correlation matrix and heatmap visualising 
  return relationships across the sector

### Key findings:
- Tesco was the strongest risk-adjusted performer with a 
  Sharpe ratio of 0.66 and 14% annual return
- Sainsbury's and Tesco showed the highest correlation (0.63), 
  reflecting shared exposure to UK grocery sector dynamics
- JD Sports showed near-zero correlation with Tesco (-0.03), 
  offering genuine diversification in a combined portfolio
